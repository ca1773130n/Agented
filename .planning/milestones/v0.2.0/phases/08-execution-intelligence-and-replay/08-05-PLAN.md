<objective>
Build the frontend components for chunk results display (EXE-03), conversation branch navigation (EXE-04), and collaborative execution viewing with presence and inline comments (EXE-05).

Purpose: Completes the user-facing interface for all Phase 8 features. Users can view merged/deduplicated chunk results, navigate branched conversation trees, see who else is watching a live execution, and post inline comments on specific log lines that appear in real-time for all viewers.

Output: PresenceIndicator, InlineComment, ChunkResults, and BranchNavigator Vue components; useCollaborativeViewer and useConversationBranch composables; collaborative and conversation-branches API clients.

Research basis: Presence indicator follows SSE-based real-time pattern (08-RESEARCH.md Recommendation 4). Branch navigator enables the tree navigation UX from ContextBranch paper (arXiv:2512.13914). Comments anchored to stdout line numbers per 08-RESEARCH.md Open Question 3.
</objective>

       export const chunkApi = {
         // Trigger chunked execution for a bot
         runChunked: (botId: string, data: { content: string; max_chunk_chars?: number }) =>
           apiFetch<{ chunked_execution_id: string; bot_id: string; total_chunks: number; status: string }>(
             `/admin/bots/${botId}/run-chunked`,
             { method: 'POST', body: JSON.stringify(data) }

         getStatus: (chunkedExecutionId: string) =>
           apiFetch<{ id: string; bot_id: string; total_chunks: number; completed_chunks: number; status: string }>(

           ),
