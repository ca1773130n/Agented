<objective>
Establish the backend foundation for all 7 specialized automation bots: register predefined trigger definitions, create the FTS5 full-text search infrastructure for BOT-07 natural language log search, build the ExecutionSearchService, and expose a search API endpoint.

Purpose: Provides the data layer and trigger definitions that all subsequent bot skill files and services depend on. FTS5 with BM25 ranking (Robertson & Zaragoza 2009) enables natural language queries over execution logs without external search infrastructure.

Output: 7 predefined bot triggers seeded on startup, FTS5 virtual table with sync triggers, ExecutionSearchService, and /admin/execution-search API endpoint.

Research basis: SQLite FTS5 with porter tokenizer for BM25-ranked search (12-RESEARCH.md, Recommendation 1). Predefined trigger pattern following existing bot-security/bot-pr-review convention.
</objective>
